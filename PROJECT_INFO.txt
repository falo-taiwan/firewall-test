FALO 企業環境部署相容性評估與閘道檢測系統 - 專案內容描述 (Readme for Zip)
==========================================================================

本壓縮包（zip）包含 FALO Enterprise Deployment Assessor 專案的核心靜態網頁檔案及說明文件。
本專案為「100% 純前端、免伺服器」的架構，可直接點擊 HTML 檔案在瀏覽器中運行。

【檔案清單與功能說明】
1. index.html   : 專案入口導覽頁面，提供前往線上評估工具與操作手冊的入口。
2. demo.html    : 企業導入部署評估與閘道相容性檢測工具（主程式），支援：
                  - 跨來源 no-cors 快速連線檢測探針（測試 50+ 雲端/AI/SaaS 平台連通性）。
                  - 15+ 項瀏覽器安全指紋診斷。
                  - 雙向資料載入還原與加密報告匯出。
3. readme.html  : 使用者操作流程說明手冊，以 Mobile-First RWD 設計，方便現場展示或傳閱。
4. README.md    : 專案總體開源技術說明文檔（Markdown 格式）。
5. TEACHING_NOTES.md : 資安與網路探針技術教學講義，包含 CORS 探針原理、隱私指紋、浮水印的學術性解說。

【使用與部署說明】
- 本機運行：直接用瀏覽器打開 index.html 即可開始使用所有功能。
- 靜態部署：可直接將 index.html, demo.html, readme.html 檔案上傳至 GitHub Pages、Vercel 或任意靜態 Web 伺服器即可完成部署。
