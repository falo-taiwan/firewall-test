FALO Enterprise Deployment Assessment & Compatibility Matrix

說明手冊與操作流程說明

這是一款專為售前顧問、系統架構師與項目經理（PM）設計的標準環境相容性評估工具。協助顧問在導入 AI 代理、自動化 Workflow、企業知識庫、RPA 等方案時，迅速核對客戶端的網路安全環境、白名單放行策略與 Gateway 架構相容性。

本專案支援 100% 前端離線運行，並針對行動裝置進行了手機操作導向（Mobile-First RWD）的介面優化，非常適合顧問在企業訪談現場使用手機或平板進行快速勾選與連線檢測。

⚡ 一鍵線上評估 (GitHub Pages)

本工具已部署於 GitHub Pages，用戶可直接使用手機或電腦開啟網址：

  • 現場測試：顧問可於會議或訪談現場，直接將 demo.html 連結傳給客戶的 IT 視窗。
  • 客戶自填：客戶在公司辦公室網路環境下點擊檢測，完畢後匯出報告並發回，免除繁瑣的手動白名單排查工作。

🌟 核心功能特點

  • 📱 行動端導向 UI (Mobile-First RWD)：底部固定 Dock 導覽分頁切換，自適應玻璃卡片流替代傳統寬表格，頂部橫向滑動分類過濾。
  • ⚡ 白名單網路連線檢測 (Reachability Engine)：利用 mode: 'no-cors' 跨來源探針，在 10 秒內跑完 50+ 個熱門網域的 TCP 握手檢測，實時掌握防火牆放行狀況。
  • 🛡️ 14+ 項資安指紋診斷 (Browser Fingerprinting)：離線收集 IP、AdBlock、GPU 晶片型號、媒體設備數、觸控支援、Canvas 渲染雜湊碼等，警示靜態網頁隱私曝露風險。
  • 🔄 數據雙向交流 (Data exchange loop)：支援匯出 UTF-8 BOM 格式 CSV、JSON 以及 HTML 加密報告。導出報告包含加密 payload，可直接拖回工具還原顧問評估數據。

📖 標準協作工作流程

  1. 顧問發送網址：顧問將 demo.html 評估網址提供給客戶的資訊部門視窗。
  2. 客戶現場盲測 (網路檢測頁面)：客戶在其內部網絡中打開網頁，輸入公司名稱後點擊「一鍵檢測連線」。測試完成後點擊「產生 HTML 報告」下載回寄給顧問。
  3. 顧問載入還原 (相容矩陣頁面)：顧問在本地打開工具，點選載入客戶發回的 HTML 報告。系統會自動還原該網絡的測試數據。
  4. 顧問補充與輸出：顧問針對受阻網域（顯示為受阻 ❌）在矩陣卡片中設定備選方案或白名單申請備忘，最終再次導出決策報告交付項目建置。

🌐 顧問現場調試與內網穿透 (轉 Port 服務)

本系統為 100% 前端靜態架構，通常直接雙擊檔案或透過 GitHub Pages 即可運行。然而，若顧問在本地修改程式碼但尚未發佈，或需要將本地伺服器展示給外網客戶時，可啟動本地伺服器（例如 python3 -m http.server 8000）並透過以下 9 種轉 Port 服務進行內網穿透：

  • Cloudflare Tunnel (推薦)：執行 cloudflared tunnel --url http://localhost:8000，提供無廣告警告、極度穩定的臨時網址。
  • ngrok：執行 ngrok http 8000，老牌穩定，但免費版有訪客警告頁。
  • Localtunnel：執行 npx localtunnel --port 8000 --subdomain falo-test，支援自訂二級網域，免註冊。
  • SSH 反向隧道 (免安裝)：執行 ssh -R 80:localhost:8000 serveo.net 或 nob@localhost.run，本機不需下載任何軟體。
  • VS Code 埠轉送：直接在 VS Code「Ports」面板中點選「Add Port」輸入 8000，並將存取設為 Public。
  • 其他相容方案：亦支援 Bore (bore local 8000 --to bore.pub)、Tailscale Funnel、Pinggy 以及自建 frp 反向代理。詳細資安分析與命令請參閱講義。
